Prepare a story

Paste your edited text, name it, seal it. This page works entirely on your machine — nothing is sent anywhere. Frontmatter at the top of the pasted text (title, character, tags…) is read automatically and fills the form.

Must be the same passphrase every time. It is never stored and never recoverable.

After downloading 1. Save the story-XXXX.json file to:
C:\Users\cathe\GitHub Repositories\the-reading-room\stories

2. Open GitHub Desktop. It will show the new file in the Changes panel.

3. Type a short summary (e.g. Add Masquerade), click Commit to main, then click Push origin.

Your story will be live in the library within about a minute.
